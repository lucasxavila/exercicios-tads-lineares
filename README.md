# Atividade - Estrutura de Dados e Análise de Algoritmos: TADs Lineares em Rust
O objetivo é a implementação e análise de complexidade de Tipos Abstratos de Dados (TADs) lineares, como Vetores, Pilhas, Filas e Deques.

## 📑 Documentação de Referência
O enunciado completo com as 20 questões pode ser consultado aqui:
- [Link para o PDF da Atividade (ATVD Avaliativa 2.pdf)](./ATVD%20Avaliativa%202.pdf)

---

## 🚀 Tecnologias Utilizadas
- **Linguagem:** Rust (Edição 2021)
- **Gerenciador de Pacotes:** Cargo
- **Bibliotecas:** `rand` (para simulações aleatórias)

## 📁 Estrutura do Projeto
O código está organizado em módulos para facilitar a correção e leitura:
- `src/grupo1.rs`: Vetores e operações básicas.
- `src/grupo2.rs`: Pilhas (Calculadora RPN, Histórico, Editor).
- `src/grupo3.rs`: Filas (Simulador de Banco, Impressora, Fila Circular).
- `src/grupo4.rs`: Deques (Palíndromos e Janela Deslizante).
- `src/grupo5.rs`: Reflexão, Análise de Performance e Round Robin.

---

## ⚙️ Como Executar

### Pré-requisitos
Você precisa ter o **Rust** e o **Cargo** instalados. Se não tiver, instale através do [rustup.rs](https://rustup.rs/).

1. Clone o Repositório:
```bash
git clone https://github.com/lucasxavila/exercicios-tads-lineares.git
```
2. Entre na pasta do projeto:
```bash
cd exercicios-tads-lineares
```
3. Rodar o projeto:
```bash
cargo run
```
4. Verificar erros (opcional):
```bash
cargo check
```
